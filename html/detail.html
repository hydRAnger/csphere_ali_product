可视化的面板

1. 提供整体概览

2. 实时监控容器数量和状态

3. 全局Docker主机和容器的健康信息展示

4. 可详细查看某个容器的配置/健康/关联Docker主机等信息


Docker容器管理

1. 集中监控所有的Docker容器

2. Docker容器的分组/分类功能

3. 清晰、直观的展现容器状态，如运行、僵尸、停止

4. 可查看容器和Docker主机的依赖关系

5. 容器的操作：启动、停止、暂停、恢复


Docker镜像管理

1. 每台Docker主机上的镜像列表

2. 快速查看镜像和容器的关系

3. 对比相同服务的镜像版本是否一致

4. 镜像的删除、push推送、tag命名操作

5. 查看镜像的历史构建过程


Docker主机管理

1. 管理Docker主机(物理机/虚拟机)容量

2. 检测Docker主机是否超载

3. Docker主机的CPU和内存信息及使用量

4. 展示Docker主机中的容器和本地镜像

5. Docker主机上的容器显示和操作: 启动、停止、暂停、删除


镜像仓库

1. 支持添加镜像仓库

2. 公有仓库、企业私有仓库


便捷的搜索

1. 可搜索Docker主机、容器和镜像

2. 可根据不同的字段过滤查找

3. 定位僵尸容器
